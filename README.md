# To-Do Application

A full-stack To-Do web application built with Laravel and PHP.
The application allows users to manage tasks with authentication, validation, and RESTful API functionality.

---

## Features

* User Authentication & Authorization
* Laravel Sanctum Authentication
* CRUD Operations for Tasks
* Form Request Validation
* RESTful APIs
* Email Verification
* Soft Deletes
* Notifications
* Events & Listeners
* Middleware & Policies
* Eloquent Relationships & Eager Loading

---

## Technologies Used

* Laravel
* PHP
* MySQL
* Laravel Sanctum
* Blade
* Bootstrap
* Git & GitHub
* Postman

---

## Installation

```bash
git clone git@github.com:abdallahhefzy05-hub/To_do_app.git

cd To_do_app

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve
```
